题解 P3924 康娜的线段树

挺不错的一道题，代码很简单

推推式子：

ans=qwq×∑i=1nval[i]2dep[i]ans = qwq \times \sum_{i = 1} ^ {n} \frac { val[i] } { 2^{dep[i]} } ans=qwq×i=1∑n​2dep[i]val[i]​

分母不同，通分一下

ans=qwq×∑i=1nval[i]×2mxdep−dep[i]2mxdepans = qwq \times \sum_{i = 1} ^ {n} \frac { val[i] \times 2 ^ {mxdep - dep[i]} } { 2 ^ {mxdep} } ans=qwq×i=1∑n​2mxdepval[i]×2mxdep−dep[i]​

先考虑没有修改

记一个ansansans，答案就是ans×qwq2mxdep\frac {ans \times qwq} {2 ^ {mxdep}}2mxdepans×qwq​

ans=∑i=1nval[i]×2mxdep−dep[i]ans = \sum_{i = 1} ^ {n} val[i] \times 2 ^ {mxdep - dep[i]} ans=i=1∑n​val[i]×2mxdep−dep[i]

ansansans可以模拟建树时求出

再考虑修改

以样例第一个第一个操作为例

可以发现，叶节点增加相当于把整个从根节点到叶节点的路径全部增加。

那么对每个叶节点维护一下根节点到叶节点的深度（黄字）的和，再求一遍前缀和，记为t[i].sumt[i].sumt[i].sum。

现在看蓝字，发现将1~3全部加4，等于把那一片蓝色全部增加。

一般地， 对于每一个修改操作(,xy,z)(,x y, z)(,xy,z)，和增加了(t[y].sum−t[x−1].sum)×z(t[y].sum - t[x - 1].sum) \times z(t[y].sum−t[x−1].sum)×z

注意：

  • 分母和乘的qwq约分

  • 不开快读会T

Code：

#include <bits/stdc++.h>
using namespace std;
#define int long long

const int N = 2333333;

struct T
{
    int sum;
} t[N << 2];

int n, m, qwq, mxdep = 1, ans;

void build(int dep, int l, int r)
{
    if(l == r)
    {
        t[l].sum = mxdep / dep * ((dep << 1) - 1);
        int a = read();
        ans += a * t[l].sum;
        t[l].sum += t[l - 1].sum;
        return;
    }

    int mid = (l + r) >> 1;
    build(dep << 1, l, mid), build(dep << 1, mid + 1, r);
}

signed main()
{
    n = read(), m = read(), qwq = read();
    
    while(mxdep < n) mxdep <<= 1; // 和dep一样，其实表示次幂
    build(1, 1, n);
    int d = __gcd(mxdep, qwq); mxdep /= d, qwq /= d;

    for(int i = 1; i <= m; ++i)
    {
        int x = read(), y = read(), z = read();
        ans += (t[y].sum - t[x - 1].sum) * z;
        cout << ans * qwq / mxdep << endl;
    }

    return 0;
}
