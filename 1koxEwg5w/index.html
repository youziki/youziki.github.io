题解 11.3 模拟赛

2020-11-04 4 min read # 比赛

P1800 software

题意：

A，B两个工作，每个工作m个模块，n个人，每个人做A模块速度a[i]，做B模块速度b[i]，所有人同时做求最短时间

n,m,a[i],b[i]<=100n,m,a[i],b[i] <= 100n,m,a[i],b[i]<=100

Solution：

考场上只有暴力O(n5logn)O(n^5logn)O(n5logn)做法，能水60tps，正解O(n3log)O(n^3log)O(n3log)，二分肯定是要的，关键在于如何dpdpdp判断是否可做完

令f[i][j]f[i][j]f[i][j]表示前i个人，做j个A，最多能做的B的个数，如果f[n][m]>=mf[n][m]>=mf[n][m]>=m说明有解，转移枚举第i个人做多少A即可，贪心的思路：已知一个人做多少A后，令他做的B最多

Code：

#include <bits/stdc++.h>
using namespace std;

const int N = 205;

int n, m, a[N], b[N], f[N][N];

inline bool check(int mid)
{
	memset(f, 0, sizeof f);
	
	int tot = 0;
	for(int i = 1; i <= n; ++i)
	{
		int J = mid / a[i];
		for(int j = 0; j <= J; ++j)
		{
			int K = (mid - j * a[i]) / b[i];
			for(int k = j; k <= min(tot + j, m); ++k)
				f[i][k] = max(f[i][k], f[i - 1][k - j] + K);
		}
		
		tot += J;
		if(f[i][m] >= m) return 1;
	}
	
	return 0;
}

signed main()
{
	scanf("%d%d", &n, &m);
	for(int i = 1; i <= n; ++i) scanf("%d%d", a + i, b + i);
	
	int l = 1, r = m * a[1] + m * b[1];
	while(l < r)
	{
		int mid = (l + r) >> 1;
		if(check(mid)) r = mid;
		else l = mid + 1;
	}
	
	printf("%d", l);
	
	return 0;
}

总结：

判断不一定要让A和B的数量刚好等于m，只要不少于即可，所以可确定一维然后贪心

dpdpdp的状态也很不错

U139126 最大后缀值个数

题意：

给一颗有点权的树，对每个点求出从根到它路径上点权等于后缀最大值的点的数量

Solution：

答案等于单调栈内元素数量是挺显然的，树上单调栈思路也不难，把回溯时还原整明白了就行了，但考场上回溯还原也没整对（可能我那样复杂度有假，不知为什么还会WA两个）

正解给出了处理回溯还原的更简单更妙的方法，不模拟单调栈，直接二分插入位置，记录下原top和这个位置原值，最后只还原这两个值即可。这样做为什么是对的？因为实际上就改变了这两个值对吧，归纳地想，其实应该弹栈的元素都没有动，而后面递归的时候后面的元素虽说被覆盖，也会在回溯的时候还原，所以其实也只有二分到的位置和top有改动

Code：

#include <bits/stdc++.h>
using namespace std;

const int N = 2333333;

vector <int> son[N];
int n, v[N], ans[N], stk[N], top;

void dfs(int x)
{
	int l = 0, r = top;
	while(l <= r)
	{
		int mid = (l + r) >> 1;
		if(stk[mid] < v[x]) r = mid - 1;
		else l = mid + 1;
	}
	int cur = stk[l], curtop = top;
	stk[l] = v[x], top = l;
	ans[x] = l + 1;
	
	for(int i = 0; i < son[x].size(); ++i) dfs(son[x][i]);
	stk[l] = cur, top = curtop;
}

signed main()
{
	ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
	
	cin >> n;
	for(int i = 2, fa; i <= n; ++i) cin >> fa, son[fa].push_back(i);
	for(int i = 1; i <= n; ++i) cin >> v[i];
	dfs(1);
	
	for(int i = 1; i <= n; ++i) cout << ans[i] << ' ';
	
	return 0;
}
