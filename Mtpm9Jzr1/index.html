9月月赛T3 [Cnoi2020]线形生物

Solution：

期望题总是式子很简单，但很玄学。。。

该题很好是因为以下几点：

  • 题里给了提示呀！——“线性生物”

    又注意到实际上这个不是一般的图上期望，题里给了是1->2->...->n+1的，所以线性推即可

  • 怎么推呢？考虑已知走到i点期望，从i点走到i+1点需要多走多少步

    除法？显然不是！

    因为概率一样，所以把所有情况的步数加起来即可！

  • 如果走到之前的点，步数怎么算？

    结果显然是之前的点再走到i点，再走一步到这个点，所以只需找出之前点走到之后点的步数即可

  • 怎么找呢？

    f数组的前缀和！

上代码！

int id,n,m,f[1000010],s[1000010];//i到i+1 1到i+1
vector<int>son[1000010];
signed main(){
	read(id),read(n),read(m);
	for(re int i=1,u,v;i<=m;++i)read(u),read(v),son[u].push_back(v);
	for(re int i=1;i<=n;++i){
		f[i]=1;
		for(re int j=0;j<son[i].size();++j)
			(f[i]+=(s[i-1]-s[son[i][j]-1]+1+mod)%mod)%=mod;//-1是因为按照定义这样才是到这个点的路径前缀和
		s[i]=(s[i-1]+f[i])%mod;
	}
	return printf("%lld",s[n]),0;
}
