题解 CF1443A 【Kids Seating】

2020-11-03 2 min read # 题解

构造

前言：

这个构造题还不算太水，如果没有选的数<=4×n<=4\times n<=4×n的限制，直接n个质数水过去了。。。

Solution：

令所有数的gcdgcdgcd为2（最简单情况），于是范围变为<=2×n<=2\times n<=2×n，观察样例：

4
8 10 12 14

在/2后为4 5 6 7，于是发现第一个数不就是n！猜到结论：

答案为（/2意义下）：n,n+1,n+2+...+2×n−1n,n+1,n+2+...+2\times n-1n,n+1,n+2+...+2×n−1

最后统一乘上2即可

Code：

#include <bits/stdc++.h>
using namespace std;
#define int long long

const int N = 1;

signed main()
{
	int t;
	cin>>t;
	while(t --)
	{
		int n;
		cin>>n;
		for(int i = 1; i <= n;++i) cout << (n + i - 1) * 2 << ' ';
		cout<<endl;
	}
	
	return 0;
}
