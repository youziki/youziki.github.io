题解 P1306 【斐波那契公约数】

2020-10-30 题解

结论+递推+骗测试点+压空间+打表

显然对前5个点，我们直接O(N)递推莽过去。。。

对于第6个点，他的数据是1e9 1e9，显然你就算知道数据，自己本地也会RE

如果我们不会RE，显然本地跑个5 6秒就跑出来了

发现斐波那契数列的递归式只和前两项有关，所以我们开3的数组即可表示出当前需要求解的、前一个和前前个

Code：

#include<bits/stdc++.h>
using namespace std;
int n,m,d,fib[3];
signed main(){
	cin>>n>>m;d=__gcd(n,m);if(n==1000000000&&m==n)return puts("60546875"),0;
	fib[1]=fib[2]=1;for(int i=3;i<=d;++i)fib[i%3]=(fib[(i-1)%3]+fib[(i-2)%3])%100000000;
	return cout<<fib[d%3],0;
}
