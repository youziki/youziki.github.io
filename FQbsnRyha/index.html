题解 CF1445A 【Array Rearrangment】

2020-11-03 题解

贪心结论严谨证明

没有别的限制，只要ai+bi<=xa_i+b_i<=xai​+bi​<=x，显然是ai+bia_i+b_iai​+bi​越小越好，判断最小的最大值是否小于等于xxx即可。

如何求最小的最大值呢？二分？太麻烦了，CF构造题很多，想是否能构造出来一种最大值最小的aaa与bbb的排列方式，不妨现将aaa升序排列，考虑如何组合bbb，最终的答案一定满足：如果bbb交换两项，答案一定不会更优，即最大值不会变小，即：

max(ai+bi,aj+bj)<=max(ai+bj,aj+bi)max(a_i+b_i,a_j+b_j)<=max(a_i+b_j,a_j+b_i)max(ai​+bi​,aj​+bj​)<=max(ai​+bj​,aj​+bi​)

考虑让这个式子不成立，不妨令i<ji<ji<j，于是ai<=aja_i<=a_jai​<=aj​，所以ai+bi<=aj+bia_i+b_i<=a_j+b_iai​+bi​<=aj​+bi​且ai+bj<=aj+bja_i+b_j<=a_j+b_jai​+bj​<=aj​+bj​，所以左式的最大值应让它为aj+bja_j+b_jaj​+bj​且大于aj+bia_j+b_iaj​+bi​，于是就有：

bj>bib_j>b_ibj​>bi​时不成立，即如果令bbb非严格降序排序，答案最优（最大值最小）

有了构造方法，即可模拟（实际上就是两个排序）求出最小的最大值，与x比较得出答案

Code：

#include <bits/stdc++.h>
using namespace std;
#define int long long
int a[23333],b[23333];

signed main()
{
	int t;
	cin >>t;
	while(t --)
	{
		int n,k,flag=1;
		cin>>n>>k;
		for(int i=1;i<=n;++i)cin>>a[i];
		for(int j=1;j<=n;++j)cin>>b[j];
		
		sort(a+1,a+n+1);sort(b+1,b+n+1);reverse(b+1,b+n+1);
		for(int i=1;i<=n&&flag;++i)if(a[i]+b[i]>k)flag=0;
		puts(flag?"Yes":"No");
		
		for(int i=1;i<=n;++i)a[i]=0;
		for(int j=1;j<=n;++j)b[j]=0;
	}
	
	return 0;
}
