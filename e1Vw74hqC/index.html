假在了哪里？如果前面段很长，它多于1的数对答案的贡献最多为1，但算入了全部，修改为一段一段模拟处理即可，没难度

T3 淘淘蓝蓝之幻影树

将幻影神奇宝贝反过来即可与从Pi到根的节点车轮战，显然与题目等价（因为原来能成功现在一定能成功，所以现在成功不了证明原来成功不了）

考虑朴素暴力：模拟车轮战，怎么卡成O(qn)O(qn)O(qn)？让它前面一直能成功，到最后（成功或不成功均可），又因为sum其实很小，所以实际上每组给出的Ki很小，又因为m很小（都为几十的范围），所以会有大量被同一个幻影神奇宝贝消灭的普通神奇宝贝

考虑优化掉这一点：预处理出每个节点到根路径第一个能消灭第i个幻影神奇宝贝的节点，预处理是O(nm)O(nm)O(nm)的，每次询问时，对于每个幻影神奇宝贝对复杂度的贡献最多为1（跳一次），所以总复杂度O(nm+sum)O(nm + sum)O(nm+sum)，代码难度不大

总结：思路不难，但没想到，看到长题面不要害怕，不要犯懒！看到sum很小，应想到复杂度转化为由sum决定的，即每个点最多处理一次（常数次）

T4 淘淘蓝蓝之电电鼠大战御坂美琴

算法标签：背包

Solution：

考虑怎样将W对复杂度的影响消去，有时，我们可以取模，但这题可以吗？考虑有所有物品体积都为n的约数的性质，考虑对n取模。观察到每一个物品都可组成成k×n+rk \times n + rk×n+r的形式，所以答案一定比W小于一个小于n的非负整数

令f[i]f[i]f[i]表示%n意义下i是否能够取到（0/1），最后在i取0到m中且f[i]f[i]f[i]为1的最大值即可，代码很好，有注释

Code：

#include <bits/stdc++.h>
using namespace std;
#define int long long

const int N = 1233333;
const int M = 65;

int n, m, cnt, sum;
int w[M], k[M], f[N], g[N], now[N];

signed main()
{
    ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);

    cin >> n >> m >> cnt; // init
    for (int i = 1; i <= cnt; ++i) cin >> w[i];
    for (int i = 1; i <= cnt; ++i)
    {
        cin >> k[i];
        if(k[i] <= n / w[i]) continue;
        sum += k[i] / (n / w[i]) - 1; // 记一下总和
        k[i] = n / w[i] + k[i] % (n / w[i]); // 只取它有意义的部分，这里多取了一个n / w[i]，相应地sum中少加了一个
    }
    
    f[0] = 1; // dp
    for (int i = 1; i <= cnt; ++i)
    {
        if(! k[i]) continue;
        memset(now, 0, sizeof now); // now表示当前模数是否可以
        for(int j = 0; j < w[i]; ++j) now[j] += f[j], g[j] = f[j];
        for(int j = w[i]; j <= 2 * n * cnt; ++j)
        {
            now[j % w[i]] += f[j];
            // 可以通过j转移而来（如果j - j % w[i]大于k[i]呢？这种情况会在之前减去，看下面）
            g[j] = now[j % w[i]] > 0;
            if(j - k[i] * w[i] >= 0)
                now[(j - k[i] * w[i]) % w[i]] -= f[j - k[i] * w[i]]; 
            // 如果j - j % w[i]大与k[i]，只要不是通过若干k[i] * w[i]相加得来的即可
            // 实时将k[i] * w[i]的贡献删去，即可保证正确
        }
        swap(f, g);
    }

    int ans = 0; // calc
    for(int i = 0; i <= 2 * n * cnt; ++i)
        if(f[i] && i <= m)
            ans  = max(ans, i + min(sum * n, (m - i) / n * n));
    cout << ans;

    return 0;
}
