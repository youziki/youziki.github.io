CF1043D Mysterious Crime

题意：

给定m个长为n的排列，求给所有排列去掉一段前缀和后缀之后使得所有排列均相同的方案数。

去掉的前缀后缀长度可以为0

保留的区间长不能为0

m<=10,n<=106m <= 10, n <= 10^6m<=10,n<=106

Solution：

难点在于n的范围太大了，暴力KMP肯定不行。做法肯定是找出所有极长的每个排列都有的子段，然后统计答案，具体地，一个长度为lenlenlen的子段的贡献为len×(len+1)2\frac len\times (len+1) 2el​n×(len+1)2

如何找极长子段呢？因为每个数只会出现一次，所以可以记录第一个排列每个数后一个是什么，然后对每个排列找每个位置的数是否能扩展，如果能，在这个数上++cnt

最后扫一遍第一个排列所有数，如果cnt == m，说明能扩展，累加当前段长，如果为falsefalsefalse，计算前一段长度的贡献

Code

Code1：
#include <bits/stdc++.h>
using namespace std;
#define int long long
const int N = 100005;
int n, m, a[20][N];
struct node{
    int nxt, cnt;
} s[N];
signed main(){
    scanf("%d%d", &n, &m);
    for(int i = 1; i <= m; ++i)
		for (int j = 1; j <= n; ++j)
			scanf("%d", &a[i][j]);
    for (int i = 1; i <= n - 1; ++i) {
        s[a[1][i]].nxt = a[1][i + 1];
        s[a[1][i]].cnt = 1;
    }
    for (int i = 2; i <= m; ++i)
        for (int j = 1; j <= n - 1; ++j)
            if(s[a[i][j]].nxt == a[i][j + 1]) ++ s[a[i][j]].cnt;
    int sum = 1, ans = 0;
    for (int i = 1; i <= n; ++i)
        if(s[a[1][i]].cnt != m) ans += sum * (sum + 1) / 2, sum = 1;
        else sum ++;
    cout << ans << endl;
    return 0;
}
