左偏树 1

2020-11-02 4 min read # 学习笔记

左偏树概述：

功能：

又叫“可并堆”，是支持合并的堆，常用操作：

插入一个数O(logn)O(logn)O(logn)

查询最小数O(1)O(1)O(1)

合并两个左偏树O(logn)O(logn)O(logn)

删除最小数O(logn)O(logn)O(logn)（实质上是合并两个左偏树）

通常可与并查集结合使用（如下面左偏树模板），并查集的根就是左偏树的根（保证相同），可以查询特定点所在的左偏树的信息

左偏树可删除任意元素，但过于复杂，很少用（不会~）

算法：

维护一个dist[]dist[]dist[]表示深度，深度定义为：两个儿子中深度较小值+1

左偏树的合并操作的复杂度与深度成正比

保证左儿子的深度大于等于右儿子的深度（因此叫左偏树）

合并时，权值小的为根，另一个树递归与右子树合并。如果回溯后左儿子深度小于右儿子深度，互换左右儿子

新建节点时，dist[]dist[]dist[]记得初始化

代码：

只有mergemergemerge一个函数，将两个（左偏树的）根节点合并，返回新根节点

int merge(int x, int y)
{
    if(! x || ! y) return x + y;
    if(cmp(y, x)) swap(x, y); // 保证x小
    rs[x] = merge(rs[x], y);
    if(dist[rs[x]] > dist[ls[x]]) swap(rs[x], ls[x]);
    dist[x] = dist[rs[x]] + 1;
    return x;
}

模板 左偏树

题意：

维护一个左偏树（不一定是一个），初始序列为空，四种操作：

  • 加入一个点作为新堆（给定点权）

  • 将第x个插入的数和第y个插入的数所在的堆合并（可能两点本身在一个堆中，忽视此操作）

  • 输出第x个插入的数所在的集合的最小数

  • 删除第x个插入的数所在的集合的最小数

Solution：

并查集处有些细节：

  • 我们在删除一个点时，不在并查集中删除它（它并不会对后面产生影响），但要让左右儿子合并的新根（权值较小的那个）在并查集中指向自己（当然要让原根和原根另一个儿子指向它），这样可以保证该点不会访问到别的点

  • 因为不能让0作为根节点，但0权值最小，所以先将0的权值赋为INF

Code：

#include <bits/stdc++.h>
using namespace std;

const int N = 233333;

int fa[N], dist[N], v[N], ls[N], rs[N], tot;

int find(int x)
{
    return x == fa[x] ? x : fa[x] = find(fa[x]);
}

inline bool cmp(int x, int y) // x是否小于y
{
    return v[x] == v[y] ? x < y : v[x] < v[y];
}

int merge(int x, int y)
{
    if(! x || ! y) return x + y;
    if(cmp(y, x)) swap(x, y);
    rs[x] = merge(rs[x], y);
    if(dist[rs[x]] > dist[ls[x]]) swap(rs[x], ls[x]);
    dist[x] = dist[rs[x]] + 1;
    return x;
}

signed main()
{
    ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
    
    int n;
    cin >> n;
    v[0] = 2e9;
    while(n --)
    {
        int t;
        cin >> t;
        if(t == 1)
        {
            int a;
            cin >> a;
            v[++ tot] = a;
            fa[tot] = tot;
            dist[tot] = 1;
        }
        else if(t == 2)
        {
            int x, y;
            cin >> x >> y;
            x = find(x), y = find(y); // 保证左偏树根节点就为并查集根节点
            if(x == y) continue;
            if(cmp(y, x)) swap(x, y); // y小就交换
            fa[y] = x; // 根节点为小的
            merge(x, y);
        }
        else if(t == 3)
        {
            int x;
            cin >> x;
            printf("%d\n", v[find(x)]);
        }
        else
        {
            int x;
            cin >> x;
            x = find(x);
            if(cmp(ls[x], rs[x])) swap(ls[x], rs[x]);
            fa[x] = rs[x], fa[rs[x]] = rs[x];
            merge(rs[x], ls[x]);
        }
    }
    
    return 0;
}
