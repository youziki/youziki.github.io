我的sb错误们

2020-10-30

1

看这一份代码：

#include<bits/stdc++.h>
using namespace std;
signed main(){
	int q;cin>>q;
	while(q--){
		int n,m;cin>>n>>m;
		int L=m,R=m,flag=0;
		for(int i=1,t=0,l,r;i<=n;++i){
			int lst=t;cin>>t>>l>>r;
			L-=t-lst,R+=t-lst;
			if(L>r||R<l){puts("NO");flag=1;break;}
			L=max(L,l),R=min(R,r);
		}if(flag)continue;
		puts("YES");
	}
	return 0;
}

它的错误在哪里？

你一组数据还没读入完就break了，叫后面的数据都读入错的！

问题在哪里？

如果你一组数据判出错，可能会输出多次“NO”，显然有问题
