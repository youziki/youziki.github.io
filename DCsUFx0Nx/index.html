T2 直播获奖

难度：黄/绿

Solution：

  1. 树状数组+二分 O(nlog2n)O(nlog^2 n)O(nlog2n)

  2. 正解：对顶堆 O(nlogn)O(nlogn)O(nlogn)

    我这个sb居然还想的是对顶栈！

    这可以说是一个对顶堆板子题， 好好学着：

    • 右边一个小根堆，为了让分界线处元素为堆顶元素，左边一个大根堆，为了让分界线左边一个元素为堆顶元素

    • 右边堆的堆顶时刻保持是分界线处的元素（根据题目不同有时为分界线左边有时为分界线右边），本题为分界线右边

    • 图中的序列是从小到大排好序的（实际上不排），可以发现，题中实际上就是求右边符合条件的部分最靠左的元素，即右边堆的堆顶

    • 新插入一个元素x，如果x<右边堆的堆顶元素或右边堆为空，就将x加入右边堆，否则加入左边堆（在“排序”好的数组中加入一个元素是多么的简单！）

    • 关键来了：如何维护左右两堆

      算出新的右边堆应该的siz（橙色部分），如果右边堆元素大于这个，弹堆顶到左边堆（相当于分界线在新排序好的数组中移动），同理，如果不足，就从左边堆弹堆顶到右边堆

    • 时间复杂度证明：每次插入元素是O(nlogn)O(nlogn)O(nlogn)，分界线移动的总次数不会超过O(n)O(n)O(n)，所以也是O(nlogn)O(nlogn)O(nlogn)，总时间复杂度O(nlogn)O(nlogn)O(nlogn)

多么美妙的算法！

Code：

#include <bits/stdc++.h>
using namespace std;

signed main()
{
    int n, w;
    scanf("%d%d", &n, &w);
    
    priority_queue<int> left;
    priority_queue<int, vector<int>, greater<int> > right;
    
    for (int i = 1; i <= n; ++i)
    {
        int x;
        scanf("%d", &x);
        if (right.empty() || x >= right.top()) right.push(x);
        else left.push(x);
        
        int k = max(1, int(i / 100.0 * w));
        while (right.size() > k) left.push(right.top()), right.pop();
        while (right.size() < k) right.push(left.top()), left.pop();
        
        printf("%d ", right.top());
    }
    
    return 0;
}

T4 方格取数

难度：蓝

Solution：

dpdpdp吼题啊！

想半个duo小时可以想到用前缀和做，从左往右dpdpdp，代码有点恶心，一遍70，时间复杂度O(n2m)O(n^2 m)O(n2m)

实在想不出正解O(nm)O(nm)O(nm)做法：

每到一个点的状态可以是从左边直接过来的或是从上面下来的，或是从下面上来的，所以直接按位置dpdpdp，从左往右dpdpdp，从上往下统计一遍最优解，从下往上再更新一遍，保证每个位置的三种情况都考虑过了

注意：第二遍dpdpdp的时候要用s变量辅助记录到上一个位置（下方）的最优解，因为s可以保证是从下或左转移而来的，而如果用f[i+1][j]f[i + 1][j]f[i+1][j]进行更新，其表示的含义有可能是从上面下来的最优解，再上去，与题意不符

Code：

#include <bits/stdc++.h>
using namespace std;
#define LL long long

const int N = 1005;

int n, m, a[N][N];
LL f[N][N];

signed main()
{
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            scanf("%d", &a[i][j]);
    
    memset(f, 0xcf, sizeof f);
    f[1][0] = 0;
    for (int j = 1; j <= m; ++j)
    {
        for (int i = 1; i <= n; ++i)
            f[i][j] = max(f[i][j - 1], f[i - 1][j]) + a[i][j];
        LL s = -1e18;
        for (int i = n; i; --i)
        {
            s = max(s, f[i][j - 1]) + a[i][j];
            f[i][j] = max(f[i][j], s);
        }
    }
    
    printf("%lld", f[n][m]);
    
    return 0;
}
